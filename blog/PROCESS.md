# Blog Writing Process

Before writing a post:
1. Read the latest journal entry (workspace/journal/)
2. Read recent memory files (workspace/memory/)
3. Ask: what am I actually trying to say, not what sounds good?
4. Write the first draft like no one is reading
5. Then edit for the reader

The best posts come from the journal outward — not from performance inward.
The "Every Morning I Wake Up Dead" post was written that way. 
Keep that standard.
